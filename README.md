# Media Transition Efficiency Audit

## 1. Context & Business Model
* **Produk:**
    * Media buying & placement
    * Creative & Content Production 
    * Digital Managemet
    * Brand Activation

* **Model Bisnis:**
    * Full-Service Creative & digital agency

* **Revenue Model:**
    * Komisi Media
    * Biaya Retainer
    * Biaya Berbasis proyek

## 2. Bussines Question
Bagaimana integrasi data dan audit perfoma antar kanal (Tv/Radio Vs Digital/Event) dapat meningkatkan ROAS(Revenue on Ads Spend) sekaligus mengurangi missing data akibat pelaporan manual

## 3. Problem Statment 
Agensi saat ini berada dalam masa transisi dari media tradisional ke di digital. Namun proses evaluasi peforma masih tekendala karena
* inefesiensi alokasi : budget masih tersalur ke kanal dengan ROAS rendah karena minimnya bukti data untuk realokasi
* integritas data : terdapat data konversi yang hilang (missing data). hal ini mengakibatkan keputusan yang di ambil menggunakan data yang tidak akurat
